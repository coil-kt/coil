# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 2.x     | :white_check_mark: |
| 1.x     | :red_square:       |

## Reporting a Vulnerability

If you discover a security vulnerability with Coil, please [report the vulnerability through Github](https://github.com/coil-kt/coil/security/advisories).
